## About
A CPU scheduling simulator built in C implementing 6 scheduling 
algorithms with Discrete Event Simulation engine.

## Algorithms Supported
- FCFS, LCFS, SRTF, Round-Robin, PRIO, PREPRIO

## Build
gcc -Wall -o osproject osprojectFinal.c

## Run
./osproject -sF input.txt rands.txt
